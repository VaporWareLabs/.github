## About VaporWare Labs

VaporWare Labs is a personal engineering lab focused on designing and building:

- 🤖 Robotics platforms
- 🚁 Autonomous drones
- 🏠 Smart home systems
- 🔌 Embedded electronics
- 🖨️ 3D-printed mechanical designs
- 🐧 Linux- and Raspberry Pi-based systems

The goal is simple:

> Build cool things, learn continuously, and document the process.

---

## Featured Projects

### 🚁 Quad-001
Endurance-focused quadcopter based on CubePilot and ArduPilot.

- S500 frame
- CubePilot flight controller
- RFD900 telemetry
- Long-range mission planning and tuning

### 🚙 Rover-001b
Modular Raspberry Pi + Arduino rover platform.

- 2020 aluminum extrusion chassis
- RC and autonomous control
- Camera and vision processing
- Future docking and charging

### 🏠 Project Vapor
A Home Assistant–based smart home ecosystem.

- Voice satellites
- Presence-aware lighting
- Scene controllers
- ESP32 nodes
